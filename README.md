# Triple Espresso Specialty Coffee Shop Website

# Description and Functionality

Coffee shop website will serve Menu and table Reservation System.

Customer can see different recipes of coffee.
Aeropress recipe
French press recipe

Customer can contact the coffee shop or see its address.
(200 Success Avenue, The Town of Triple Peaks)

Customer can see hourse of operations.
Hours:
Monday – Friday (10:00 – 19:00)
Saturday – Sunday (11:00 – 18:00)

Customer can book a table for guests. In the Book a table form, customer can specify the number of guests, time and date of booking and other contact information.

In Menu section, a detailed menu is added in the website where the customer can see the available drinks and baked items with prices.

About section provides the information about the free wifi and other facilities at coffee shop.

Footer shows the social media links (deactive) and the author name with copyright.

# Technologies and Techniques Used

HTML and CSS used in the website.

BEM (Block Element Modifier) methodology used for organising the HTML structure and also the files.

Flexbox and Positioning relative and absolute used to arrange the elements.

iframe tag used for embeding videos from youtube.com.

Forms are used to acquire orders and to book a table.

Different types of transformation applied like scale.

Animations via keyframes are used to pulsate the circle on the about page.

Transition Property used to eliminate the jarring effects while hover over the button or link.
